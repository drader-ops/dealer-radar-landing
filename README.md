# 📡 딜러 레이더 - 랜딩페이지

> 중고차 딜러를 위한 AI 통화 분석 서비스 **딜러 레이더**의 전환율 중심 랜딩페이지

---

## 🚀 완성된 기능

### 📄 페이지 구조 (6단계 후킹 구조)
| 섹션 | 설명 |
|------|------|
| **Hero** | 강한 헤드라인 + 서브카피 + CTA 버튼 + 파티클 애니메이션 |
| **Problem** | 딜러 공감 포인트 3가지 카드 |
| **Solution** | 3단계 솔루션 플로우 (화자 분리 → 시그널 포착 → 코칭) |
| **Demo** | 인터랙티브 데모 플레이어 (실시간 타이핑 애니메이션) |
| **Features** | 4가지 핵심 기능 카드 그리드 |
| **Results** | 카운터 애니메이션 통계 + 후기 카드 |
| **Final CTA** | 베타 신청 폼 |

### 🎮 인터랙션 기능
- **AI 데모 플레이어**: 재생 버튼 클릭 시 가상 대화가 실시간 타이핑 효과로 표시
- **구매 시그널 태그**: 대화 중 예산/선호/망설임 시그널이 실시간으로 감지
- **AI 영업 코칭**: 데모 완료 후 구체적인 후속 영업 전략 제안
- **상담 평가 점수**: 애니메이션 스코어 바 표시
- **웨이브폼 시각화**: 재생 진행에 맞춰 오디오 파형 업데이트

### ✏️ 편집 모드
- 우측 하단 **"편집 모드"** 버튼 클릭으로 활성화
- 파란 점선 테두리가 표시된 텍스트를 직접 클릭해서 수정 가능
- **저장** 버튼으로 로컬스토리지에 변경사항 보존
- 편집 가능한 요소: 모든 헤드라인, 서브카피, 기능 설명, 통계 레이블, 후기 등

### 🎨 디자인/UX
- 다크 테마 SaaS 스타일 디자인 (네이비/블루 계열)
- 모바일/태블릿/데스크톱 완전 반응형
- 스크롤 reveal 애니메이션
- 숫자 카운터 업 애니메이션
- 히어로 파티클 효과
- Sticky 네비게이션 + 스크롤 시 배경 변화
- 햄버거 모바일 메뉴

## 🔧 커스터마이징 가이드

### 텍스트 수정 방법 (두 가지)

**방법 1 - 편집 모드 (UI):**
1. 페이지 우측 하단 `✏️ 편집 모드` 버튼 클릭
2. 파란 테두리 텍스트 직접 클릭 후 수정
3. `💾 저장` 버튼 클릭

**방법 2 - HTML 직접 수정:**
`index.html`에서 `data-editable="키이름"` 속성이 있는 요소를 찾아 수정

### 주요 수정 포인트 (HTML)

| `data-editable` 값 | 수정 위치 |
|---------------------|-----------|
| `hero-headline` | 히어로 메인 헤드라인 |
| `hero-sub` | 히어로 서브카피 |
| `problem-title` | 문제 섹션 제목 |
| `problem-1~3` | 문제 카드 3개 |
| `solution-title` | 솔루션 제목 |
| `sol-1/2/3-title/desc` | 솔루션 3단계 |
| `features-title` | 기능 섹션 제목 |
| `feat-1/2/3/4-title/desc` | 기능 카드 4개 |
| `results-title` | 기대 효과 제목 |
| `stat-1/2/3` | 통계 레이블 |
| `testimonial-1/2` | 후기 텍스트 |
| `cta-headline` | 최종 CTA 헤드라인 |
| `cta-sub` | 최종 CTA 서브텍스트 |

### 색상 변경 (`css/style.css` 상단 `:root`)
```css
--primary: #2563EB;        /* 메인 블루 색상 */
--accent: #F59E0B;         /* 포인트 골드 색상 */
--bg-dark: #0A0F1E;        /* 배경 다크 색상 */
```

### 데모 스크립트 변경 (`js/main.js`)
`DEMO_SCRIPT` 배열을 수정하여 대화 내용 변경 가능
`COACHING_TEXT` 변수로 AI 코칭 멘트 변경 가능
`SCORE_DATA` 배열로 평가 점수 변경 가능
